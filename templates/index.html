Hikuri API
¿Quiénes Somos?
Hikuri es una plataforma innovadora que brinda acceso a una extensa base de datos de plantas medicinales utilizadas en la herbolaria tradicional mexicana.
Con Hikuri, los desarrolladores pueden acceder fácilmente a información detallada sobre una amplia variedad de hierbas, incluyendo sus nombres comunes y científicos, usos terapéuticos, preparaciones recomendadas.
